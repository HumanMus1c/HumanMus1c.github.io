#🔲安装hexo博客框架

npm install -g hexo-cli

#初始化博客

npx hexo init

#🔳安装Next主题(主题文件放置到theme文件夹下)

npm install hexo-theme-next

#本地预览网站步骤

#每次修改文件都要重新生成博客页面

hexo c
hexo g
hexo s

#📝设置主题并引用theme config配置文件

theme: next
theme配置信息next要和根目录下themes文件夹中放置的子主题文件夹名一致，才会读取主题文件以及配置信息。

#🚀配置部署信息

github仓库名格式：<github ID>.github.io

1.将默认分支改为:main🔧
